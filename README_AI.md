## Контекст для ИИ

Полная документация и правила проекта находятся в папке `docs/`.

### Начать отсюда

1. `docs/agent-onboarding.md`
2. `docs/README.md` (оглавление)

### Самые важные правила

- **NO_DB_MIGRATIONS**: никаких миграций SQLite — базы пересоздаются через Admin.
- **Тестирование**: только через MCP tools на подключённом сервере (см. `docs/testing-protocol.md`).
